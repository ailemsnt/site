# PORTFÓLIO WEB 

## Descrição do projeto: 
Este projeto foi desenvolvido como primeira página web criada, a partir dos conhecimentos adquiridos no curso de Desenvolvimento de Software - Carreira Tech do SCTec. 
O portfólio foi desenvolvido utilizando **HTML5, CSS e Javascript**, seguindo boas práticas de layout com responsividade.

Consiste em uma **landing page responsiva** contendo as seguintes informações:
- Seções:
  - **Início:** - Apresentação pessoal com foto.
  - **Sobre mim:** - Descrição de habilidades comportamentais, formação acadêmica, tecnologias e habilidades técnicas.
  - **Projetos:** - Cards exemplificando alguns projetos reais desenvolvidos em tecnologias distintas, com imagem e descrição.
  - **Contato:** - Formulário funcional, com validação nos campos digitáveis, preparação para envio de e-mail e feedback visual através de alertas.

## Tecnologias utilizadas:
- **HTML5** - estrutura semântica da página;
- **CSS** - Estilização do conteúdo com **flexbox, grid, animações, responsividade, menu hamburger**;
- **Javascript** ES6 - Funcionalidades dinâmicas:
  - Acionamento de _menu hamburguer_ (três linhas)
  - Validação de campos do formulário utilizando _regex_
  - Contador automático de dias até o próximo ano
  - Preparação para envio de e-mail
- **Google fonts** - tipografia "Quicksand" e "Anton".
- **Visual Studio Code** - IDE de desenvolvimento
- **Canva** - planejamento e design da página
